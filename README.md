# RAG_sandbox
This is a playground for a RAG system with streamlit interface

# Usage

* Make sure to rename .env.sample to .env and populate it with your vars
* Run `uv sync`
* Run `uv run streamlit src/RAG_app.py`
* Upload PDF file(s)
* Enable RAG and optionally reranker, set params for retriever
* Have fun :) Observe the context precision metric
